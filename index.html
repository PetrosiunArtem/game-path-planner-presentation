Progress Path Planner

Проблема

  • Гайды по играм не персонализированы: не учитывают ваши навыки, оружие, текущий прогресс
  • Игроку сложно понять, сколько времени займёт цель и какой порядок шагов даст лучший шанс.

Цель проекта

Создать персональный планировщик прогресса для игр, который помогает игроку достигать конкретных целей с прогнозом времени и количества попыток.

Функциональные требования

  • Ввод профиля: [список доступных оружий, навыки, список побежденных боссов, статус уровней]

  • Обновление прогресса вручную: [отметить пройденный уровень, купленное оружие и тд]

  • Получение двух альтернативных маршрутов для конкретной цели

Сценарии использования часть 1

Пользователь регистрирует свой прогресс по игре, ставит цель победить конкретного босса и получает два варианта событий:

  1. Потратить время на сбор допустимых ресурсов
  2. Купить эффективное оружие против данного босса
  3. Изучить паттерны поведения босса и тактики против него

  1. Купить навыки, специализирующиеся на уклонении
  2. Практиковать парирование и уклонения и играть на выносливость

Описание API (REST)

Base: /api/v1

  • GET /game/cuphead/goals — список целей
  • POST /game/cuphead/plan — расчёт плана по профилю
  • PATCH /profiles/{id}/progress — прогресс уровней/покупок
  • POST /telemetry/boss_attempt — факт попытки босса
  • POST /telemetry/run_level — факт прохождения Run‑and‑Gun
  • GET /content/version — версия контента

Модель времени и попыток

  • Босс: фазы HP / эффективный DPS (оружие × accuracy) + оверхед.
  • psuccess = sigmoid(a0 + a1·survival + a2·parry + a3·movement + a4·familiarity − a5·difficulty).
  • Ожидаемое: (T_raw + θ·T_raw·(1 − p)) / p, где θ≈0.5 — среднее время до смерти.
  • Run‑and‑Gun: T = T_base·(1 − movement·γ)·(1 − familiarity·δ) / p_success.

Безопасность и ToS

  • Без чтения памяти/инжекта. Ручной ввод/файл сейва (по желанию, с согласия).
  • Источник данных: публичные вики и speedrun‑метрики.
  • Телеметрия по времени — опт‑ин, минимум персональных данных.

Бизнес‑модель и метрики

  • Freemium: базовый план бесплатно; PRO — «советник лоадаута», тренировки и аналитика.
  • Метрики: точность ETA (MAPE), доля завершений планов, снижение попыток, D7 ретеншн.

План MVP (2 недели)

  1. Сбор HP/фаз боссов, базового DPS оружий, маршрутов монет.
  2. JSON‑граф 20–40 действий, оценщик времени/попыток.
  3. Планировщик A* + Монте‑Карло; 2–3 альтернативы.
  4. UI: профиль → план → альтернативы → чеклист → пересчёт.
