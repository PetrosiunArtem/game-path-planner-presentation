Game Path Planner

Персональный план к игровой цели с оценкой времени

Фокус: одна конкретная игра (замените Игра X на вашу)

Автор: @PetrosiunArtem

Приложение строит оптимальный маршрут к цели в игре, учитывая прокачку, знание карты и инвентарь. Показывает альтернативы и прогноз времени.

Проблема → Решение

  • Гайды и вики не персонализированы под ваш прогресс и стиль.
  • Непонятно, сколько времени займёт цель.
  • Лишние шаги из‑за незнания карты.
  • Персональный план из действий к цели.
  • 2–3 альтернативных маршрута с ETA и рисками.
  • Учет прокачки, инвентаря и «знания карты».

Почему лучше существующих

  • Персонализация маршрутов и времени.
  • Прозрачность: видно вес каждого шага.
  • Актуальность: версии под патчи игры.

Целевая игра

Игра X (замените на нужную), платформа: ПК/консоль

MVP поддерживает одну игру, далее — расширение.

Пользовательские сценарии

  1. Быстрый план к цели: выбрать цель → заполнить профиль → получить 3 маршрута с ETA.
  2. Пересчёт после прогресса: отметить сделанное → уточнить время.
  3. Сравнение путей: «быстрее» vs «надёжнее» vs «ресурсосберегающий».

Архитектура (MVP)

  • Граф действий (узлы/рёбра) с весами: время/риск/ресурсы.
  • Планировщик A*/Дейкстра с модификаторами игрока.
  • Неопределённость: min/median/p95 + Монте‑Карло.

Данные: вики/гайды и краудсорс времени.

REST API (черновик)

Base: /api/v1

  • GET /games/{id}/goals — список целей
  • POST /plans — расчёт плана
  • PATCH /profiles/{id}/progress — обновление прогресса
  • POST /telemetry/time — фактическое время шагов
  • GET /content/version — версия контента
  • GET /actions/{id} — карточка действия

Данные и калибровка

  • Эталон: вики/гайды/спидран-сводки.
  • Краудсорс: пользователи подтверждают время.
  • Храним min/median/p95; применяем коэффициенты игрока.

Юридика и безопасность

  • Без чтения памяти процесса (античит).
  • Импорт сейва офлайн и по согласию, минимум данных.
  • Соблюдение ToS, публичные источники.

Бизнес‑модель и метрики

  • Freemium + PRO аналитика.
  • DLC‑пакеты под сложные цели/боссы.
  • MAPE ETA, завершения планов, D7 ретеншн.

План MVP (2–4 недели)

  1. Одна игра, 2–3 цели, граф 20–40 действий.
  2. Профиль → расчёт → альтернативы → чеклист.
  3. Сбор фактического времени.
