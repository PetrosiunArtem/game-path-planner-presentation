# Game Path Planner — презентация на Reveal.js

Статическая презентация для защиты идеи MVP. Сделана на [Reveal.js](https://revealjs.com/) с CDN, поэтому хостится как обычная веб‑страница.

## Как запустить локально
Открой `index.html` в браузере.  
Навигация: стрелки ← → или Space. Номера слайдов — внизу.

Печать в PDF: добавьте к URL `?print-pdf` и распечатайте из браузера.

## Публикация на GitHub Pages
1. Залей файлы в репозиторий.
2. В Settings → Pages выбери **Branch: main** и **/ (root)**.
3. Открой `https://<твой-юзер>.github.io/<имя-репо>/`.

## Что нужно адаптировать под вашу игру
- На слайдах замените «Игра X», названия целей и примеры действий.
- При необходимости добавьте скриншоты игры (вставляйте в отдельные слайды).
- В API‑секцию добавьте реальные поля вашего сейва/перков.

## Структура
- `index.html` — презентация.
- `styles.css` — лёгкий "wireframe" набор для макетов.
